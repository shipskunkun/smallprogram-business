## 5-3 小程序组件名称、路径

组件的名字不取决于目录，取决于usingComponents 中设置的 key值

起名规范，不同团队不一致  
比如写个like组件  
有的 v-like   
有的 like-cmp  
和团队命名习惯保持一致即可


```
//key 目录
"usingComponents": {
    "v-like": "components/like/index"
}

// 代表和该文件同级目录
比如这个文件为 pages/classic/classic.json 
v-like 的路径就为  pages/classic/components/like/index
```
